CGB

API bancaire et application de gestion des virements liée au traitement des remboursements de l'application GSB.

Présentation du projet

API bancaire développée en Java avec Spring Boot, simulant l'interface bancaire pour le traitement des remboursements de frais de l'application GSB.

Cette API gère l'authentification des paiements, la validation des transactions et le suivi des remboursements des visiteurs médicaux.

Fonctionnalités clés

Authentification : Sécurisation avec Spring Security et JWT.
Audit : Journalisation complète des transactions.
Documentation : Swagger pour tester l'API.
Qualité : Développement piloté par les tests (TDD).
Traitement : Remboursement des virements par lots.

Technologies utilisées

Java
Spring Boot
Spring Security
H2 Database
JWT
Swagger
OAuth2
