Portfolio Personnel

Réalisation d’un portfolio web dans le cadre de mon BTS SIO SLAM, afin de centraliser mes projets, parcours et compétences.

Présentation du projet

Le portfolio a été développé afin de regrouper l’ensemble de mes travaux réalisés durant ma formation BTS SIO option SLAM. Il a été pensé comme un support à la fois personnel et professionnel, pour illustrer mes compétences et expériences.

Le site a été construit en HTML, CSS et JavaScript, avec une attention portée à la structure, à la lisibilité, à l’ergonomie.

La Réalisation

Organisation : Conception d’une structure HTML sémantique, hiérarchisation claire du contenu.
Design : Utilisation de CSS personnalisé avec des composants réutilisables, effets de transition, et gestion du responsive.
Fonctionnalité : Intégration d’un menu hamburger interactif, formulaire de contact connecté à EmailJS, et affichage dynamique des projets.

Technologies utilisées

HTML
CSS
JavaScript
EmailJS
GitHub
