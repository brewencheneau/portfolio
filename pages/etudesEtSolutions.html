Études et Solutions Réseau / OpenWRT / Raspberry Pi

Ensemble de missions techniques réalisées autour du déploiement réseau, Raspberry Pi, OpenWRT, scripts Python et interface Django dans le cadre du BTS SIO SLAM.

Présentation du projet

Cette série de missions avait pour objectif de concevoir un schéma réseau sécurisé, configurer un OS spécialisé sur Raspberry Pi, développer un scanner réseau et déployer une application web embarquée avec des outils comme Django et Python. Ces missions ont permis de renforcer les compétences en sécurité, administration réseau, développement embarqué et automatisation.

Fonctionnalités clés

• Étude des solutions d'accès sécurisé à distance (Reverse Proxy, VPN, Tailscale...)
• Configuration d’OpenWRT sur Raspberry Pi
• Déploiement de l’application web via Git sur des Raspberry Pi segmentés
• Développement de scripts de démarrage automatique (batch + Python)
• Création d’un scanner réseau interactif avec Django et base SQLite
• Interface d’administration et intégration dans un projet existant
